# 🧠 Feelio — AI가 읽어주는 나만의 감정 기록

> 감정을 기록하면 AI가 당신의 마음을 읽어드립니다.

**Feelio**는 사용자가 일상의 감정을 자유롭게 기록하면, AI가 이를 분석하여 감정 패턴을 시각화하고 맞춤형 케어를 제공하는 감정 다이어리 서비스입니다.

## 🚀 핵심 기능

- **감정 일기 작성** — 텍스트로 오늘의 감정을 자유롭게 기록
- **AI 감정 분석** — OpenAI 기반 감정 분류 및 인사이트 제공
- **실시간 케어** — 분석 결과에 따른 맞춤형 응원 메시지
- **감정 기록 조회** — 과거 기록을 돌아보며 감정 변화 추적
- **주간 리포트** — 일주일간의 감정 흐름을 차트로 시각화

## 🛠️ 기술 스택

| 영역 | 기술 |
|---|---|
| Frontend | React, Chart.js / Recharts |
| Backend | Node.js, Express |
| Database | MongoDB |
| AI | OpenAI API |
